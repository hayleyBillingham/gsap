PixiJS Plugin

  • PixiJS is a canvas library that can utilize WebGL for insanely fast rendering
  • PixiJS has all sorts of crazy filters and fun effects
  • You could always use GSAP to animate PixiJS Objects
  • It can be a tad cumbersome with certain properties because the're tucked inside sub-objects in PixiS API
  • PixiJS defines rotatioanl values in radians instead of degrees which isnt as intutitve for developers
  • PixiPlugin saves you the headaches
